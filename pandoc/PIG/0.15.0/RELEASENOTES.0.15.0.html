Apache Pig 0.15.0 Release Notes

These release notes cover new developer and user-facing incompatibilities, important issues, features, and major improvements.

  • PIG-4519 | Minor | Correct link to Contribute page

Correct link to Contribute page

  • PIG-2692 | Minor | Make the Pig unit faciliities more generalizable and update javadocs

Added the ability to mock multiple aliases when running a job. Added assertOutputAnyOrder to be order independent. Added user docs for mocking.

  • PIG-2647 | Major | Split Combining drops splits with empty getLocations()

Don't ignore unavailable blocks when combining input splits into PigSplits.
